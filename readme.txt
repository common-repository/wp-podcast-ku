WP Podcast KU adalah plugin sederhana untuk membuat dan menampilkan podcast player dari file audio.

== Description ==

WP Podcast KU adalah plugin sederhana untuk membuat dan menampilkan podcast.

Dengan WP Podcast KU, Anda bisa dengan mudah membuat audio player yang ciamik serta menarik.

Selain itu terdapat 2 tipe akses dari Plugin ini, yakni
- publik
- member only (user harus login terlebih dahulu)

Yang artinya, Anda bisa juga memonesisai podcast Anda dengan mudah tanpa menggunakan pihak ketiga.
